Common Questions

  • How do I register to vote?

    If you have a valid ID from the DMV, you can register online. Otherwise, see full instructions here.

  • What do I need to bring when I vote?

    Any kind of ID -- a photo ID is not required.

  • What if they say

    If you believe you are at the poll site assigned to your home address, ask for an “affidavit” ballot.

    Note that your vote will not be counted if you vote at a poll site that is not tied to where your registered home address is, so double check this using NY's Voter Lookup Tool.
